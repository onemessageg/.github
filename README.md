# OneMsg

![OneMsg Hero](.github/profile/assets/banner_hero.png)

## Overview
OneMsg is a next-generation, cross-platform messaging ecosystem built for security, high-performance, and a seamless native experience across iOS, Android, Web, and Desktop.

## Repositories
- [**OneMsg iOS**](https://github.com/onemessageg/ios) - Native Swift/SwiftUI mobile application.
- [**OneMsg Android**](https://github.com/onemessageg/android) - Native Kotlin/Jetpack Compose mobile application.
- [**OneMsg Web**](https://github.com/onemessageg/web) - Modern React-based web and desktop experiences.
- [**OneMsg Backend**](https://github.com/onemessageg/backend) - High-scale Go microservices and infrastructure.
- [**OneMsg Docs**](https://github.com/onemessageg/docs) - Technical documentation and community resources.

## Features
- **End-to-End Encryption**: Secure messaging by default.
- **Cross-Platform Sync**: Seamless transition between devices.
- **Advanced Social**: Threads, Polls, and rich media stories.
- **Integrated Commerce**: Crypto and fiat wallet for global payments.
- **Collaborative**: Real-time co-browsing and voice-to-text accessibility.
